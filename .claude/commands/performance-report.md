Generate a comprehensive performance report for the current quarter.

This will:
1. Load all competency scores and progress
2. Include quarterly question responses (if evaluated)
3. Highlight key achievements
4. Identify areas for improvement
5. Export to markdown file

The report is suitable for sharing with your manager during quarterly reviews.

What format would you like?
- **markdown** (default) - For review and editing
- **json** - For programmatic use
