# Archived Scripts

This directory contains one-time migration scripts that are no longer needed
for regular operation but are kept for reference.

## Scripts

- `backfill_q1_2026.py` - Backfill data for Q1 2026
- `fix_session_projects.py` - One-time fix for session project data
- `migrate_state.py` - State migration from old format
- `migrate_to_config_dir.sh` - Config directory migration

These scripts should NOT be run unless you understand their purpose and have
verified they are needed for your specific situation.
